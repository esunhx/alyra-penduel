# Penduel

This excericise is an attempt to simulate a blockchain-based hangman game.

## Run & Test

Start the react dev server.

```sh
$ cd client
$ npm start
```

Build the truffle project

```sh
$ cd truffle
$ truffle compile
```

Migrate smart-contracts

```sh
$ cd truffle
$ truffle migrate
```

Run the smart-contracts test

```sh
$ cd truffle
$ truffle test
```
